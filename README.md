# 🍀 Hello Nature 🍀
  
## 프로젝트 개요
  * 온라인 푸드테크 기업 헬로네이처의 웹사이트 제작

## 프로젝트 구성원
  * 김민지
  * 김성용
  * 박소민
  * 이수지
  * 이예솔
  * 정영훈

## 프로젝트 목적 및 필요성

  - 헬로네이처 웹사이트는 앱에 최적화 되어있어 웹을 이용하는 소비자에게 불편을 줄 수 있음. 따라서 소바자가 웹 사이트를 
  이용하면서 겪을 불편함을 고려하여 앱에 최적된 사이트를 웹버전으로 개발 하였음 

## 프로젝트 진행과정

  * 스토리 보드 
  * 테이블정의서
  * API문서
  * 디자인 및 퍼블리싱
  * 더미데이터를 활용한 프론트 서버 및 백엔드 서버 구현
  * 오류 점검 및 최종 코드 수정
  * 발표
